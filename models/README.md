# GGUF Models Directory

Place your .gguf model files here for inclusion in exported builds.

## Recommended Models

Download one of these models for development:

- **SmolLM-135M** (~145MB) - Testing/prototyping
- **Qwen2.5-0.5B-Instruct** (~530MB) - Recommended for production  
- **TinyLlama-1.1B-Chat** (~670MB) - Higher quality

Use the test scene to download models: addons/ohmydialog/examples/llama_test_scene.tscn

## Note

Models downloaded during development are stored in user://models/
Copy your chosen model here (res://models/) for export builds.
